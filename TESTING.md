# 测试说明

## 已覆盖

`LiuliuFriendsTests/GameViewModelTests.swift` 覆盖：

- 每轮只有一个正确选项
- 1-2 岁玩法每轮最多两个候选
- 点对后记录完成状态
- 点错后进入重试反馈
- 下一轮切换
- 重置进度回到第一轮
- 会话基础暖场顺序
- 会话轮次数量按年龄段递减，并控制单次会话总量，防止高阶玩法或大题库主导单次体验
- 前 8 次完成保持低龄玩法，之后解锁更高阶玩法
- 启蒙阶段选择会过滤超龄玩法，并在当前题超出阶段时回到可用题
- 家长设置玩法开关分组覆盖所有玩法，避免新增玩法后漏出或混入错误层级
- 点错后再答对的题目会延迟复现，多个复习题之间会穿插普通题，重置进度会清空复习队列
- 多少朋友会显式验证“多的/少的”正确项数量关系，避免只靠固定答案凑题
- 用途、场景、天气、作息、情绪、动作、触感、搭配、相反、找不同、节奏、顺序、规律等抽象关系题都有显式目标配置，避免只靠物体名称凑题
- 颜色目标 ID 保持唯一，避免颜色玩法在单次题库里重复同一读音目标

## Mac 上运行

```bash
xcodebuild test -project LiuliuFriends.xcodeproj -scheme LiuliuFriends -destination 'platform=iOS Simulator,name=iPad (10th generation)'
```

如果本机模拟器名称不同，先查看可用设备：

```bash
xcrun simctl list devices available
```

## 人工验收清单

- iPad 横屏：主游戏页没有文字或按钮重叠
- iPhone 竖屏：主要按钮仍然可点，文字没有截断到不可读
- 点对：星星、音效、六六开心状态正常出现
- 点错：候选轻微摇动，不出现失败惩罚
- 设置页：四个开关能即时影响游戏反馈
- 家长设置：家长信息、隐私、网络、广告和内容说明正常展示
- 无网络：运行时不弹系统权限请求
- iPad 横屏：全屏运行，不触发方向支持警告
