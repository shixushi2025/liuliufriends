# 肚兜游戏 - 六六找朋友

这是一个 SwiftUI iOS/iPadOS 原型项目，面向 1-2 岁婴幼儿。

## 定位

- 系列品牌：肚兜游戏
- 首款游戏：六六找朋友
- 目标设备：iPad 优先，兼容 iPhone
- 目标年龄：1-2 岁

## 第一版玩法

游戏采用单屏循环，不做分数、失败、倒计时。

每次会话会先从动物、声音、颜色、形状做基础暖场，开局前 8 次完成只在 18m+/24m+ 的低龄玩法内轮换；之后再从各玩法中限量抽取题目并均衡打乱，避免动物、声音、大小等题库数量大的模式连续出现太久。点错后再答对的题目会隔几题温和复现一次，帮助孩子巩固，不显示“错题”或惩罚提示。

1. 动物朋友：帮六六找到指定动物。
2. 声音朋友：听声音，点出对应动物或常见物体。
3. 颜色朋友：通过气球、皮球、书本、杯子、花朵、雨伞等形态找到同颜色物体。
4. 形状朋友：认识圆形、方形、三角形、星星、爱心等基础形状。
5. 大小朋友：帮小伙伴找到同样大小的家。
6. 影子朋友：根据轮廓找到对应朋友。
7. 数量朋友：找出一、二、三、四个相同物体，作为数字认知入口。
8. 分类朋友：从动物、水果、车辆、生活物品中找出同一类朋友。
9. 位置朋友：认识上面、下面、左边、右边等基础空间方位。
10. 用途朋友：根据喝水、看书、遮雨、乘坐等生活用途找到对应物品。
11. 场景朋友：根据海里、天空、路上、家里、花园、雨天等场景找到合适朋友。

## 当前功能完整性

- 首页游戏循环：已完成，包含基础暖场、前 8 题低龄自适应、全玩法均衡轮换和错题温和复习

- 家长设置：已完成，包含设置开关、隐私、网络、广告和内容说明；玩法开关按基础识物、观察匹配、理解关系分组

- 测试：已加入 `LiuliuFriendsTests`
- 正式素材：已接入 App 图标、六六、小猫小狗和常见动物/水果/车辆/物品图片

## 角色设定

- 系列品牌：肚兜游戏
- 主角：六六
- 造型：穿红肚兜的圆圆小宝宝，偏玩偶感，不做真实婴儿
- 性格：好奇、开心、爱找朋友
- 反馈：点对时开心拍手/跳跳，点错时温和提醒，不批评孩子
- 图标方向：六六大头 + 红肚兜，不在图标内放完整游戏名

## 运行

用 Xcode 打开 `LiuliuFriends.xcodeproj`，选择 iPad 或 iPhone 模拟器运行。

推荐验证命令：

```bash
xcodebuild test -project LiuliuFriends.xcodeproj -scheme LiuliuFriends -destination 'platform=iOS Simulator,name=iPad (10th generation)'
```

在 `LiuliuFriends/Views/CharacterViews.swift` 里有两个 Xcode Preview：

- 六六角色：普通状态和开心状态
- 图标概念：六六头像 + 红肚兜的 App 图标方向
