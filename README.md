# 肚兜游戏 - 六六找朋友

这是一个 SwiftUI iOS/iPadOS 原型项目，面向 1-2 岁婴幼儿。

## 定位

- 系列品牌：肚兜游戏
- 首款游戏：六六找朋友
- 目标设备：iPad 优先，兼容 iPhone
- 目标年龄：1-2 岁

## 第一版玩法

游戏采用单屏循环，不做分数、失败、倒计时。

每次会话会先从动物、交通、水果、声音、颜色、形状做基础暖场，开局前 8 次完成只在 18m+/24m+ 的低龄玩法内轮换；之后再按年龄段限量抽取题目并均衡打乱，18m+ 玩法保留更多练习，30m+/36m+ 抽样更少，同时单次会话按当前玩法数量保留一轮覆盖，避免新增玩法后某些玩法被挤出。点错后再答对的题目会隔几题温和复现一次，帮助孩子巩固，不显示“错题”或惩罚提示。

1. 动物朋友：帮六六找到指定动物。
2. 交通朋友：认识汽车、公交车、火车、卡车、飞机、小船、自行车、消防车、救护车、拖拉机、火箭等交通工具。
3. 水果朋友：认识苹果、香蕉、橙子、梨、草莓、西瓜、葡萄、桃子、菠萝、樱桃、柠檬等常见水果。
4. 声音朋友：听声音，点出对应动物或常见物体。
5. 颜色朋友：通过气球、皮球、书本、杯子、花朵、雨伞等形态找到同颜色物体。
6. 形状朋友：认识圆形、方形、三角形、星星、爱心等基础形状。
7. 身体朋友：认识眼睛、耳朵、嘴巴、小手、小脚、鼻子等常见身体部位。
8. 衣物朋友：认识帽子、衣服、裤子、鞋子、袜子、外套等常见穿戴物。
9. 蔬菜朋友：认识胡萝卜、玉米、番茄、黄瓜、蘑菇、西兰花等常见蔬菜。
10. 食物朋友：认识米饭、面条、面包、鸡蛋、牛奶、饼干等日常食物。
11. 餐具朋友：认识小碗、勺子、盘子、叉子、筷子、奶瓶等吃饭用具。
12. 卫生朋友：认识牙刷、牙膏、毛巾、香皂、澡盆、梳子等干净习惯物品。
13. 家居朋友：认识椅子、桌子、小床、沙发、台灯、时钟等家里物品。
14. 文具朋友：认识铅笔、蜡笔、橡皮、尺子、本子、书包等学习用品。
15. 乐器朋友：认识小鼓、钢琴、吉他、喇叭、铃铛、麦克风等会发声的物品。
16. 玩具朋友：认识积木、玩偶、风筝、拼图、摇铃、小桶等常见玩具。
17. 自然朋友：认识太阳、云朵、月亮、花朵、小树、彩虹等自然物。
18. 地点朋友：认识家、学校、公园、海边、商店、游乐场等常见地点。
19. 职业朋友：认识医生、老师、警察、消防员、厨师、农民等常见职业。
20. 大小朋友：帮小伙伴找到同样大小的家。
21. 长短朋友：比较长长的和短短的物体，认识横向长度关系。
22. 高矮朋友：比较高高的和矮矮的物体，认识竖向高度关系。
23. 影子朋友：根据轮廓找到对应朋友。
24. 数字朋友：认识 1-10 的数字卡片，听“找数字三”后选择对应数字。
25. 数量朋友：找出一、二、三、四个相同物体，作为数量认知入口。
26. 多少朋友：比较多的和少的，作为数量理解的进阶入口。
27. 分类朋友：从动物、水果、蔬菜、食物、车辆、身体、衣物、餐具、卫生、家居、文具、乐器、玩具、自然、地点、职业、生活物品中找出同一类朋友。
28. 位置朋友：认识上面、下面、左边、右边等基础空间方位。
29. 里外朋友：认识里面、外面，找到在盒子里或盒子外的朋友。
30. 前后朋友：认识前面、后面，找到在前面或后面的朋友。
31. 用途朋友：根据喝水、看书、遮雨、乘坐等生活用途找到对应物品。
32. 场景朋友：根据海里、天空、路上、家里、花园、雨天等场景找到合适朋友。
33. 同处朋友：根据小鱼、海边、桌子、学校等线索，找到常在同一个地方出现的朋友。
34. 天气朋友：根据晴天、雨天、多云、有风等天气线索找到对应物体。
35. 作息朋友：根据早上、喝水、看书、玩耍、出门、睡觉等日常作息找到对应物体。
36. 情绪朋友：根据开心、安静、加油等表情提示找到对应的六六角色状态。
37. 动作朋友：根据会飞、会游、会滚、会跳等动作线索找到对应朋友。
38. 触感朋友：根据软软、硬硬、滑滑、粗粗等触感词找到对应物体。
39. 材料朋友：根据木头、金属、纸、玻璃、布、塑料等材料线索找到对应物体。
40. 味道朋友：根据甜甜、酸酸、多汁、淡淡等味道词找到对应食物或物体。
41. 搭配朋友：根据蜜蜂和花朵、小猴和香蕉、雨伞和云朵、职业和工具等生活关联找到好搭档。
42. 动物家朋友：根据海里、鸟窝、蜂窝、洞洞、农场、池塘等居住线索找到对应动物。
43. 动物宝宝：根据猫妈妈、狗妈妈、鸭妈妈、牛妈妈、羊妈妈、马妈妈等亲子线索找到对应动物宝宝。
44. 动物食物：根据小兔、小猴、小猫、小鸭、蜜蜂、小马等动物线索找到它爱吃的食物。
45. 物品回家：根据书包里、书架上、刷牙杯里、门口、玩具盒里、餐桌上等放置线索找到对应物品。
46. 相反朋友：根据白天/夜晚、上面/下面、快快/慢慢等相反关系找到对应提示。
47. 不同朋友：根据同类样本找到不一样的物体。
48. 节奏朋友：根据拍拍手、踏踏脚、摇一摇、敲一敲等节奏动作找到对应提示。
49. 顺序朋友：根据第一个、第二个、最后一个等前后顺序找到对应序列卡。
50. 规律朋友：根据 A-B-A-? 的简单规律找到下一个朋友。

- 测试：已加入 `LiuliuFriendsTests`

## 角色设定

- 系列品牌：肚兜游戏
- 主角：六六
- 造型：穿红肚兜的圆圆小宝宝，偏玩偶感，不做真实婴儿
- 性格：好奇、开心、爱找朋友
- 反馈：点对时开心拍手/跳跳，点错时温和提醒，不批评孩子
- 图标方向：六六大头 + 红肚兜，不在图标内放完整游戏名

## 运行

用 Xcode 打开 `LiuliuFriends.xcodeproj`，选择 iPad 或 iPhone 模拟器运行。

推荐验证命令：

```bash
xcodebuild test -project LiuliuFriends.xcodeproj -scheme LiuliuFriends -destination 'platform=iOS Simulator,name=iPad (10th generation)'
```

在 `LiuliuFriends/Views/CharacterViews.swift` 里有两个 Xcode Preview：

- 六六角色：普通状态和开心状态
- 图标概念：六六头像 + 红肚兜的 App 图标方向
